== 緒言
* Autohotkey v2のスクリプト
* 「Space」などを修飾キーとして扱う
** スペースキーダウンから300ms未満（スクリプト設定による）のアップで本来のキーを入力、それ以上で何もしない。キーを押しながら他のキーを押すことで、設定された動作 
* 各キーの長押しでシフト押し動作
* 「Caps Lock」「ひらがな」は、ahkでの動作が不安定のため、レジストリでScanCodeを書き換え、F13、F14を割り当てておくこと
** レジストリを書き換えなければ、ahkファイルのF13に関する設定は無視される 
* 日本語キーボード向け作っているので、USキーボード向けには改変すべきだが、「Space」を修飾キーとして使う機能はむしろ、キーの少ないUSキーボード向け
* スクリプトは試行錯誤中のもの、かつ、特殊なキー配置を考慮しているので、カスタマイズを推奨
* 誤動作発生の可能性がある
** 動作環境依存で発生頻度が異なるが、キーリピートの開始遅くする（表示までの待ち時間を長くする）し、キーリピートを遅くする（表示の間隔を遅くする）すると改善する

== long_press_shift.ahk
=== 動作概要
* 「Space」や「F13」が修飾キー、つまり、このキーと他のキーの同時押しで特殊な動作が行われる、2つのキーはほぼ同じ役割だが、一部異なる
* 各キーの長押しでシフト押し動作、連打はできないので、そのときは従来通りShiftキーで行うこと
* F13とEscの同時押しでReloadだが同時にリセットになる、不安定なときに押すとよい
* (Space or F13)と","の同時押しでテンキー入力モードの切り替え、該当キー以外のキー押下で解除
* (Space or F13)と"."の同時押しでキーマウスモードの切り替え、該当キー以外のキー押下で解除
* F14(ひらがな)/変換/無変換も修飾キーとして動作
